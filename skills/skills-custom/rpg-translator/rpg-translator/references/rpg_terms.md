# RPG Terminology Reference (EN -> PL)

Ten plik zawiera glosariusz common RPG terms oraz propozycje tłumaczeń zależnie od gatunku.

## Podstawowe Atrybuty (Core Attributes)
- Strength -> Siła
- Dexterity -> Zręczność
- Constitution -> Budowa / Kondycja
- Intelligence -> Inteligencja
- Wisdom -> Mądrość / Roztropność
- Charisma -> Charyzma / Osobowość

## Mechanika Walki i Testów
- Hit Points (HP) -> Punkty Wytrzymałości (PW) / Punkty Życia (PŻ)
- Armor Class (AC) -> Klasa Pancerza (KP)
- Advantage / Disadvantage -> Ułatwienie / Utrudnienie (lub Przewaga / Niekorzyść)
- Proficiency Bonus -> Premia z biegłości
- Critical Hit -> Trafienie krytyczne
- Initiative -> Inicjatywa

## Gatunkowe Słowniczki

### Fantasy
- Dungeon -> Podziemia / Lochy
- Spell slots -> Komórki czarów
- Cantrip -> Sztuczka
- Long Rest -> Długi odpoczynek

### Sci-Fi / Cyberpunk
- Cyberware -> Cyberwszczepy
- Data-jack -> Złącze danych
- AI -> SI (Sztuczna Inteligencja)
- Credits -> Kredyty

### Eldritch Horror
- Sanity -> Poczytalność
- Madness -> Szaleństwo
- Clues -> Poszlaki / Ślady
- Ancient One -> Przedwieczny
